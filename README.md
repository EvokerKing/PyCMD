# PyCMD Instructions

How to get PyCMD for the first time:
1. Make sure you are running on windows (it does not support linux, and mac yet)
2. Get python (3.5 or later) if you haven't already
3. Get PIP if you haven't already
4. Navigate to the folder you downloaded
5. Run pyinstall using command prompt

How to run PyCMD:
1. Navigate to the folder you downloaded using file explorer or command prompt
2. Run pystart using cmd
